Module: LanguageStore

This store manages the current locale for internationalization (i18n). It holds the currently selected language, which can be either 'de' (German) or 'en' (English). The locale is also reflected in the global i18n instance.

(inner) setLocale(locale)

This action updates the locale state in the store and also updates the locale in the global i18n instance, ensuring synchronization across the application.

Parameters:

locale string

The new locale to set. Should be either 'de' or 'en'.
